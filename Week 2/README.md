# SOC Week 2 Task

## Overview
This project demonstrates SOC operations including alert management, incident response, alert triage, IOC validation, evidence preservation, and a capstone project.

## Tools Used
- Google Sheets
- Google Docs
- VirusTotal
- Kali Linux

## Tasks Completed
- Alert Classification and Prioritization
- Incident Ticket Creation
- Escalation Email
- Alert Triage and IOC Validation
- Evidence Collection and Hashing
- Incident Response (Capstone Project)

## Outcome
This project simulates real-world SOC workflow and improves skills in threat detection, analysis, and response.
